## :scroll: About

A discord bot built with JavaScript, NodeJS and DiscordJS. 

## :question: How to use

**1)** Clone this repo:
```elm
git clone https://github.com/felpshn/unity-bot.git

cd unity-bot
```
**2)** Setting up and running:
```elm
npm install

npm run dev
```
